Accidents and Incidents from 1908 to 2023

As soon as I finished my IBM Data Analyst Professional Certificate I wanted to start some projects to learn and develop further my skills.

Aviation.

I decided to start a small project on the history of the aviation accidents and incidents.
This project involved web scraping using Python, data cleaning, analyzing and visualizing data using various libraries such as Pandas, Matplotlib, seaborn and a final interactive dashboard using powerBI

Data Wrangling

Before we start transforming and cleaning the data, here's a brief descript of some columns:

  • Date: Date of accident, in the format - January 01, 2001
  • Time: Local time, in 24 hr
  • Airline/Op: Airline or operator of the aircraft.
  • Flight #: Flight number assigned by the aircraft operator
  • Route: Complete or partial route flown prior to the accident
  • AC Type: Aircraft type
  • Reg: ICAO registration of the aircraft
  • cn / ln: Construction or serial number / Line or fuselage number
  • Aboard: Total aboard (passengers / crew)
  • Fatalities: Total fatalities aboard (passengers / crew)
  • Ground: Total killed on the ground
  • Summary: Brief description of the accident and cause if known
