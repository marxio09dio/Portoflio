An Overview of the Steam Store Games

A game developer approached me to get insights into the Steam Store world. The goal was to understand where to best allocate resources to develop a successful game.
As a gamer myself, I was really excited for this project and was really into it, maybe because this was part of my imaganition, but I digress...

The first step was to obtain the data; for this, there were several options and different sources for each type of data.
After that, the data was analyzed, and a report was provided, including an interactive dashboard where the developer could also interact with the data.

Obtaining the Data

After exploring multiple alternatives, I came across an excellent project by nik-davis (maybe this is where I also got my inspiration). This data was excellent, but it was outdated (more than 4 years old), so, as the Python noob that I am, I decided to get my own data:

  • First, I got the list of the App IDs that are present on the Steam Store from the API URL
  • With all the IDs collected, I got to work by getting all the data from each url of the Steam store: "https://store.steampowered.com/api/appdetails/?appids=appid"
  • The The Steam Spy API was used to collect data like Positive and Negative Reviews and the Peak Players for yesterday (yes, literally the day before getting the data, not when you are reading this).
  • To get a count of the peak players of all time, well, I had to do a bit of scraping from the website . Steam Player Count
